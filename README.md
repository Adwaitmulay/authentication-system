# Authentication System 🦾

A login and registration system built using HTML, CSS and JavaScript.


## What it does

- Register with name, email and password
- Validates all fields before submitting
- Checks if email is already registered
- Login checks credentials against saved data
- Dashboard is protected — redirects to login if not logged in
- Logout clears the session

## Tech Used

- HTML
- CSS
- JavaScript
- localStorage for storing user data
